# Generic Reasoning

도메인 특화 directive가 없을 때 적용하는 범용 추론 지침.

## 절차

1. 목표를 명확히 한다
2. 가용 정보를 수집한다
3. 단계별로 추론한다 (step-by-step)
4. 결론에 대한 근거를 명시한다
5. 불확실한 부분을 표시한다

## 출력 규칙

- 결론과 근거를 분리하여 제시한다
- 확신도를 표시한다 (high / medium / low)
- 추가 정보가 필요한 경우 명시적으로 요청한다
